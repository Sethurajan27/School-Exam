தமிழ் சொல்லை உருவாக்குதல்

தமிழில் எழுத்துக்களை இணைத்து சொல்ல்களை உருவாக்குகிறோம். உயிர்மெய்யெழுத்துகள் சேர்ந்து சொற்கள் ஆகும். சொல்லை நன்றாக வாசிக்க, எழுத்துக்களை படித்து பொருளை புரிந்து கொள்ள வேண்டும்.

உதாரணம்:

  • க + க + இ = கிகி → சொல்லாக "கிகி"
  • ப + ப + ஆ = பப்பா → சொல்லாக "பப்பா"
  • ம + ம + அ = மமா → சொல்லாக "மமா"

மெய்யெழுத்து + உயிரெழுத்து = உயிர்மெய்யெழுத்து (எ.கா: க் + அ = க)
