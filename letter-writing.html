Letter Writing

Writing a letter is a way to send a message to someone far away. Letters have special parts that help us write clearly.

A letter usually has these parts:

  • Heading: Where you write the address and date.
  • Greeting: A way to say hello (like Dear Aunt).
  • Body: The main message of the letter.
  • Closing: A polite way to end the letter (like Yours truly).
  • Signature: Your name at the end.
Example Letter:

123 Main Street
May 10, 2025

Dear Grandma,

I hope you are well. I am having fun at school.
I like learning new things every day.

Yours truly,
Sara

Flowchart: Steps to Write a Letter

Think About Who → Write Heading → Write Greeting → Write Body → Write Closing → Write Signature → Finish
